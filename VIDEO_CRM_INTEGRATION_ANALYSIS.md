# تحليل شامل: ربط مقاطع الفيديو (Spotlight) مع CRM

## 📊 الوضع الحالي

### نظام Spotlight (الفيديو):
- **Collection:** `spotlight_videos`
- **VideoModel** يحتوي على:
  - `carId` - معرف السيارة (اختياري)
  - `realEstateId` - معرف العقار (اختياري)
  - `sellerId` - معرف البائع
  - `viewsCount`, `likesCount` - إحصائيات
  - `url`, `thumbnail` - روابط الفيديو

### نظام CRM:
- **المبيعات (Sales):**
  - `SaleModel` (Car): يحتوي على `carId`
  - `Sale` (Real Estate): يحتوي على `propertyId`
  
- **الإيجارات (Rentals):**
  - `RentalModel`: يحتوي على `propertyId`
  
- **العملاء (Customers):**
  - `RealEstateCustomerModel`: معرف العميل
  - `CustomerModel` (Car): معرف العميل

---

## 🔗 الربط المطلوب

### 1. **ربط الفيديوهات بالمبيعات (Sales)**

#### للعقارات:
```
Sale (propertyId) → VideoModel (realEstateId)
```

**السيناريو:**
- عند عرض عملية بيع عقار في CRM
- البحث عن الفيديوهات المرتبطة بهذا العقار (`realEstateId == propertyId`)
- عرض الفيديوهات في تفاصيل عملية البيع

#### للسيارات:
```
SaleModel (carId) → VideoModel (carId)
```

**السيناريو:**
- عند عرض عملية بيع سيارة في CRM
- البحث عن الفيديوهات المرتبطة بهذه السيارة (`carId`)
- عرض الفيديوهات في تفاصيل عملية البيع

---

### 2. **ربط الفيديوهات بعقود الإيجار (Rentals)**

```
RentalModel (propertyId) → VideoModel (realEstateId)
```

**السيناريو:**
- عند عرض عقد إيجار في CRM
- البحث عن الفيديوهات المرتبطة بهذا العقار (`realEstateId == propertyId`)
- عرض الفيديوهات في تفاصيل عقد الإيجار

---

### 3. **ربط الفيديوهات بالعملاء (Customers)**

```
Customer → VideoModel (sellerId)
```

**السيناريو:**
- عند عرض تفاصيل عميل في CRM
- البحث عن الفيديوهات التي أنشأها هذا العميل (`sellerId == customerId`)
- عرض قائمة بالفيديوهات في صفحة العميل

---

### 4. **إحصائيات الفيديوهات في Dashboard**

**البيانات المطلوبة:**
- إجمالي الفيديوهات المرتبطة بالعقارات/السيارات المباعة
- إجمالي المشاهدات للفيديوهات المرتبطة بالمبيعات
- إجمالي الإعجابات
- الفيديوهات الأكثر مشاهدة

## 📈 السيناريوهات العملية

### السيناريو 1: عرض فيديو في تفاصيل عملية البيع

```
1. المستخدم يفتح "إدارة المبيعات"
2. يضغط على عملية بيع معينة
3. النظام يبحث عن الفيديوهات المرتبطة بـ propertyId/carId
4. يعرض الفيديوهات في Tab منفصل
5. المستخدم يشاهد الفيديو مباشرة
```

### السيناريو 2: ربط فيديو بعملية بيع جديدة

```
1. المستخدم يضيف عملية بيع جديدة
2. النظام يبحث تلقائياً عن الفيديوهات المرتبطة بـ propertyId/carId
3. يعرض قائمة بالفيديوهات المتاحة
4. المستخدم يختار الفيديوهات المرتبطة
5. يتم حفظ الربط في Firestore
```

### السيناريو 3: إحصائيات الفيديوهات

```
1. المستخدم يفتح Dashboard
2. يعرض:
   - إجمالي الفيديوهات المرتبطة بالمبيعات
   - إجمالي المشاهدات
   - الفيديوهات الأكثر فعالية
   - ROI للفيديوهات
```
