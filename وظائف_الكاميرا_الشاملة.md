# 📸 وظائف الكاميرا الشاملة في التطبيق

## 🎯 نظرة عامة
التطبيق يحتوي على نظام كاميرا متقدم يدعم 3 أوضاع رئيسية للتصوير مع ميزات متعددة.

---

## 📱 أوضاع التصوير (Camera Modes)

### 1️⃣ **الوضع العادي (Normal Mode)**
- **الوصف**: تصوير صور عادية عالية الجودة
- **الاستخدام**: لتصوير العقارات والسيارات بشكل تقليدي
- **المميزات**:
  - دقة عالية (ResolutionPreset.max)
  - إضافة علامة مائية (Watermark)
  - تسجيل صوتي اختياري (3 ثواني)
  - حفظ metadata كامل (الإعدادات، الموقع، الوقت)

### 2️⃣ **الوضع البانورامي (Panorama Mode)**
- **الوصف**: إنشاء صور بانورامية 360 درجة
- **الاستخدام**: لتصوير مساحات واسعة (غرف، واجهات، حدائق)
- **المميزات**:
  - يتطلب **8 صور** من زوايا مختلفة
  - دمج تلقائي للصور
  - حجم إخراج: **4096x2048** بكسل
  - شريط تقدم يوضح عدد الصور المتبقية
  - إلغاء العملية متاح في أي وقت

### 3️⃣ **الوضع ثلاثي الأبعاد (3D Mode)**
- **الوصف**: إنشاء نموذج ثلاثي الأبعاد للعنصر
- **الاستخدام**: لتصوير السيارات والعقارات بشكل تفاعلي
- **المميزات**:
  - يتطلب **12 صورة** من زوايا مختلفة (كل 30 درجة)
  - دليل زاوية يوضح الزاوية التالية
  - شريط تقدم يوضح التقدم
  - إنشاء ملف JSON وصفي للنموذج
  - حجم إخراج: **4096x4096** بكسل

## 🛠️ الخدمات والوظائف

### 📷 **CameraService** (خدمة الكاميرا الأساسية)

#### الوظائف الرئيسية:
1. **`initialize()`**
   - تهيئة الكاميرا
   - اكتشاف الكاميرات المتاحة
   - إعداد دقة عالية
   - تفعيل/تعطيل الصوت

2. **`captureImage()`**
   - التقاط صورة واحدة
   - إضافة علامة مائية تلقائياً
   - تسجيل صوتي اختياري (3 ثواني)
   - حفظ metadata كامل
   - إرجاع معلومات الصورة (الحجم، الأبعاد، المسار)

3. **`updateConfig(CameraConfig)`**
   - تحديث إعدادات الكاميرا
   - تغيير وضع الفلاش
   - تعديل الدقة
   - تغيير إعدادات الصوت

4. **`dispose()`**
   - إغلاق الكاميرا
   - تنظيف الموارد

#### Metadata المحفوظة:
- نموذج الجهاز
- الموقع (خط الطول والعرض)
- إعدادات الكاميرا (الدقة، الفلاش، التعرض، التركيز)
- معلومات العلامة المائية
- معلومات الصوت
- اسم المستخدم
- التاريخ والوقت

---

### 🌐 **PanoramaService** (خدمة الصور البانورامية)

#### الوظائف:
1. **`startCapture()`**
   - بدء عملية التقاط بانورامي
   - مسح الصور السابقة

2. **`addImage(String imagePath)`**
   - إضافة صورة جديدة للبانوراما
   - تحديث شريط التقدم

3. **`finishCapture()`**
   - دمج جميع الصور (8 صور)
   - إنشاء صورة بانورامية نهائية
   - حفظ metadata خاص بالبانوراما

4. **`cancelCapture()`**
   - إلغاء العملية
   - مسح جميع الصور المحفوظة

5. **`cleanup()`**
   - حذف جميع الملفات المؤقتة

#### المتطلبات:
- **8 صور** على الأقل
- حجم الإخراج: **4096x2048** بكسل
- نوع الملف: JPG

---

### 🎲 **ThreeDService** (خدمة النماذج ثلاثية الأبعاد)

#### الوظائف:
1. **`startCapture()`**
   - بدء عملية التقاط 3D
   - إعادة تعيين الزاوية الحالية

2. **`addImage(String imagePath)`**
   - إضافة صورة من زاوية جديدة
   - تحديث الزاوية الحالية (+30 درجة)
   - تحديث شريط التقدم

3. **`finishCapture()`**
   - إنشاء نموذج 3D من 12 صورة
   - حفظ ملف JSON وصفي
   - حفظ جميع الصور في مجلد منظم

4. **`getNextAngleGuide()`**
   - إرجاع نص يوضح الزاوية التالية
   - مثال: "التقط 5 صور أخرى. قم بتدوير العنصر إلى 150 درجة"

5. **`cancelCapture()`**
   - إلغاء العملية
   - مسح جميع البيانات

6. **`cleanup()`**
   - حذف جميع الملفات المؤقتة

#### المتطلبات:
- **12 صورة** من زوايا مختلفة
- كل صورة بزاوية **30 درجة** (360/12)
- حجم الإخراج: **4096x4096** بكسل
- ملف JSON وصفي يحتوي على:
  - عدد الصور
  - خطوة الزاوية
  - التاريخ والوقت
  - نوع النموذج

---

### 🎨 **WatermarkService** (خدمة العلامة المائية)

#### الوظائف:
1. **`addWatermark()`**
   - إضافة نص كعلامة مائية على الصورة
   - **المعاملات**:
     - `imagePath`: مسار الصورة
     - `watermarkText`: نص العلامة المائية
     - `position`: موضع العلامة (topLeft, topRight, bottomLeft, bottomRight, center)
     - `opacity`: الشفافية (0.0 - 1.0)
     - `color`: لون النص
     - `size`: حجم الخط

#### المواضع المدعومة:
- **topLeft**: أعلى يسار
- **topRight**: أعلى يمين
- **bottomLeft**: أسفل يسار
- **bottomRight**: أسفل يمين
- **center**: الوسط

#### المميزات:
- حفظ الصورة مع العلامة المائية في ملف جديد
- عدم تعديل الصورة الأصلية
- دعم الشفافية والألوان المخصصة

---

### 🎤 **AudioService** (خدمة التسجيل الصوتي)

#### الوظائف:
1. **`hasPermission()`**
   - التحقق من صلاحيات الميكروفون

2. **`init()`**
   - تهيئة خدمة التسجيل

3. **`recordAudio()`**
   - تسجيل صوتي تلقائي لمدة **3 ثواني**
   - حفظ الملف بصيغة **M4A**
   - إرجاع مسار الملف

4. **`stopRecording()`**
   - إيقاف التسجيل

5. **`deleteRecording()`**
   - حذف الملف الصوتي

6. **`getRecordingDuration()`**
   - الحصول على مدة التسجيل

7. **`dispose()`**
   - تنظيف الموارد

#### المميزات:
- تسجيل تلقائي عند التقاط الصورة (إذا كان مفعلاً)
- حفظ في مجلد مؤقت
- دعم صيغة M4A

## 🎮 عناصر التحكم (Controls)

### **CameraControls Widget**
- **زر التقاط**: التقاط صورة
- **زر تبديل الكاميرا**: التبديل بين الكاميرا الأمامية والخلفية
- **مؤشر المعالجة**: يظهر أثناء معالجة الصورة

### **ModeSelector Widget**
- **زر الوضع العادي**: 📷
- **زر الوضع البانورامي**: 🌐
- **زر الوضع 3D**: 🎲

### **CameraOverlay Widget**
- **شريط التقدم**: للوضع البانورامي و3D
- **دليل الزاوية**: للوضع 3D
- **مؤشرات التوجيه**: تساعد المستخدم في التقاط الصور

## 📂 أماكن الاستخدام في التطبيق

### 1. **شاشة Spotlight Feed**
- زر الكاميرا في شريط التنقل
- يؤدي إلى `CameraScreen`

### 2. **إضافة العقارات**
- استخدام `ImagePicker` لاختيار الصور من المعرض
- استخدام `PropertyImageService.captureImage()` للتقاط من الكاميرا

### 3. **إضافة السيارات**
- استخدام `ImagePickerField` لاختيار الصور
- دعم الصور المحلية والصور من الكاميرا

### 4. **تعديل الملف الشخصي**
- استخدام `ImagePicker` لاختيار صورة الملف الشخصي
- دعم الكاميرا والمعرض

## 🔧 الإعدادات (CameraConfig)

```dart
class CameraConfig {
  final bool enableAudio;          // تفعيل التسجيل الصوتي
  final String watermarkText;      // نص العلامة المائية
  final Position watermarkPosition; // موضع العلامة
  final double watermarkOpacity;    // شفافية العلامة (0.0 - 1.0)
  final Color watermarkColor;      // لون العلامة
  final double watermarkSize;      // حجم الخط
  final String resolution;         // دقة الصورة
}
```

### للوضع العادي:

5. حفظ الصورة مع metadata

4. دمج الصور تلقائياً
5. حفظ الصورة البانورامية النهائية

### للوضع 3D:
1. فتح الكاميرا

4. اتباع دليل الزاوية
5. إنشاء النموذج 3D
6. حفظ الملفات والوصف
